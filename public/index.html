WhatsApp Webhook Service

Service Status: Active

About

This is a WhatsApp webhook service that integrates with Monday.com to manage incoming messages and leads.

API Endpoint

The webhook is available at /api/webhook and supports:

  • GET - Meta webhook verification
  • POST - Process incoming WhatsApp messages

Features

  • ✅ Automatic lead creation in Monday.com
  • ✅ Message history tracking
  • ✅ Contact deduplication
  • ✅ Real-time message processing
